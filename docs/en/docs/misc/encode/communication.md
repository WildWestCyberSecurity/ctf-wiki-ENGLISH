## Tap Code

Tap code is a method of encoding text messages in a very simple way. Named for the way information is encoded through a series of tapping sounds, Tap code is based on a 5×5 Polybius square, with the difference that the letter K is merged into C.

|Tap Code|1|2|3|4|5|
|--|--|--|--|--|--|
|1|A|B|C/K|D|E|
|2|F|G|H|I|J|
|3|L|M|N|O|P|
|4|Q|R|S|T|U|
|5|V|W|X|Y|Z|
