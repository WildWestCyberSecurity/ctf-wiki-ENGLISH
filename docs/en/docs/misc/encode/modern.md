# Common Encodings in the Real World

## Barcode

-   Multiple black bars and white spaces of varying widths, arranged according to specific encoding rules, used to express a set of information as a graphical identifier
-   International Standards
-   EAN-13 product standard, 13-digit number
-   Code-39: 39 characters
-   Code-128: 128 characters
-   [Online Barcode Reader](https://online-barcode-reader.inliteresearch.com/)

## QR Code

-   A data symbol that records information using specific geometric shapes arranged in a pattern of black and white on a plane according to certain rules
-   Stacked/Row-based 2D codes: Code 16k, Code 49, PDF417
-   Matrix 2D codes: QR CODE
