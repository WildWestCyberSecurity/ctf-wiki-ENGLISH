# Information Gathering Techniques

## Network Information Gathering Tips

-   Public channels
-   Target web pages, geographic locations, related organizations
-   Organizational structure and personnel, personal profiles, phone numbers, emails
-   Network configuration, security protection mechanism strategies and technical details
-   Methods for searching for specific security vulnerabilities or private information through search engines
-   [Google Hacking Database](https://www.exploit-db.com/google-hacking-database/)
-   Circumventing internet restrictions

## Basic Search Tips

-   Google basic search and mining techniques
-   Keep keywords simple and clear
-   Use words most likely to appear on the web pages you want to find
-   Describe what you are looking for as concisely as possible
-   Choose uniquely descriptive words
-   Public information database queries
-   Personal information: Census Bureau
-   Entities such as businesses: YellowPages, business credit information websites
-   Websites, domains, IPs: whois, etc.
